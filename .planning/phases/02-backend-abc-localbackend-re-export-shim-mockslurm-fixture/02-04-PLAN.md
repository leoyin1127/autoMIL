# Plan 02-04: Rename `orchestrator.py` ŌåÆ `_orchestrator_daemon.py` + Re-export Shim

## Objective

Rename the 1115-line `src/automil/orchestrator.py` to `src/automil/backends/_orchestrator_daemon.py` using `git mv` (preserving blame history), then create a 5-line backward-compat re-export shim at the original path, and update `compat.py` to record the migration. This is the structural prerequisite for Plan 02-05's `LocalBackend` ŌĆö the new `local.py` imports `ExperimentOrchestrator` from `_orchestrator_daemon`.

Per D-60, the rename is pure mechanical: no logic changes in `_orchestrator_daemon.py`. The 387-test suite is the behavioural contract ŌĆö if any test breaks, the shim is wrong. Per PATTERNS.md ┬¦5, the shim uses PEP 562 `__getattr__` for lazy deprecation warnings rather than a bare star-import (fires warning on use, not on `import automil.orchestrator`).

Purpose: Without this rename, `local.py` cannot import from the allowed-list path; the lint script (Plan 02-07) would flag `orchestrator.py` for containing `Popen`/`os.kill` at a path that is NOT in the allowlist.

Output: `_orchestrator_daemon.py` (git mv of orchestrator.py), new 5-line `orchestrator.py` shim, updated `compat.py`. All 387 tests green.

    ```python
    # Import from automil.backends instead:
    #   ExperimentOrchestrator  -> automil.backends._orchestrator_daemon.ExperimentOrchestrator
    # Will be removed in v2.0. See automil.compat for migration table.
    import warnings as _warnings
    from automil.backends._orchestrator_daemon import (  # noqa: F401, F403
        ExperimentOrchestrator,
    )
    from automil.backends._orchestrator_daemon import *  # noqa: F401, F403


    def __getattr__(name: str):
        _warnings.warn(
            f"automil.orchestrator.{name} moved to automil.backends._orchestrator_daemon "
            f"in Phase 2 (D-60). Update imports by 2027-01.",
            DeprecationWarning,
            stacklevel=2,
        )
        from automil.backends import _orchestrator_daemon as _mod  # noqa: F401
        return getattr(_mod, name)
    ```
