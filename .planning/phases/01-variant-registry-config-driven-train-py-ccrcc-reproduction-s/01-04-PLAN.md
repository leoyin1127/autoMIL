<objective>
Ship the two static validators that gate variant submissions. Plan 01-01 defined the type system; Plan 01-02 defined registration; this plan defines what makes a variant module *acceptable* for the registry.

Per D-30, three validators total — `interface`, `purity`, `identity`. Phase 1 splits them by run-time:
- **`interface`** (this plan): submit-time, **static** — walks AST + `inspect.signature`. Rejects classes that don't subclass the matching ABC OR whose required methods are missing/wrong-signature.
- **`purity`** (this plan): submit-time, **static** — pure AST walk; no module import. Rejects top-level I/O (`open`, `requests`, `urllib`), top-level network/filesystem side-effects, mutable module-level globals.
- **`identity`** (Plan 01-05, Wave 4): instantiate-time, runtime — needs torch tensors. Mode-aware (`free` vs `architecture-preserving`).

Per D-32, validator failure is **hard-fail** with no soft-warn substitute — the operator gets a file:line + fix suggestion, the experiment never queues. This defeats Pitfall 1 ("still uses old path") because a variant whose `forward` signature differs from `ModelVariant.forward` cannot pass interface, and a module that opens a network socket on import cannot pass purity.

Per D-30 specifics, `interface` MUST run before `purity` (cheap structural fail before AST walk) — but both are short-circuited on first failure with the failed validator's name in the error.

Purpose: Without these validators, a malformed variant would crash deep inside training (e.g., `AttributeError: 'CeSmooth008' object has no attribute 'forward'` 30 minutes into a 4h run). The static check at submit time fails fast — orchestrator never queues the experiment, GPU never wastes cycles.

```python
from __future__ import annotations

from dataclasses import dataclass, field
from pathlib import Path
from typing import Optional


@dataclass
class ValidationError(Exception):
    """Raised by interface/purity/identity validators on hard-fail.

    Operator-friendly: `path` + `line` + `reason` + `fix_suggestion` fields are
    plumbed through Plan 01-07's submit-hook into a click.ClickException with
    a "Refusing to submit: ... at <path>:<line>: <reason>. <fix>." message.
    """
    validator_name: str               # "interface" | "purity" | "identity"
    path: Path                        # variant module path
    line: Optional[int]               # AST line number where the violation was detected; None if module-level
    reason: str                       # one-line description
    fix_suggestion: str               # "Add `def forward(...)` to the class" etc.
    column: Optional[int] = None      # AST col_offset; None if not available

    def __str__(self) -> str:
        loc = f"{self.path}:{self.line}" if self.line else str(self.path)
        return (
            f"[{self.validator_name}] {loc}: {self.reason} "
            f"Fix: {self.fix_suggestion}"
        )
```

```python
class InterfaceValidator:
    """Static validator: ABC subclass + required-method signature check.

    The `check(module_path)` method:
      1. Parses the module via `ast.parse(module_path.read_text())`.
      2. Locates the variant class (the `@register(...)` decorator target).
      3. Reads the spec's `kind` from the decorator call's first arg.
      4. Imports the module IN A SUBPROCESS-SAFE WAY (using importlib in the
         current process is necessary to invoke the @register decorator and
         resolve the class object for signature inspection).
      5. Verifies `issubclass(cls, ABCs[kind])`.
      6. Walks the ABC's required methods via `inspect.signature(abc_method)`
         and compares to the variant class's same-named method's signature.
      7. Raises `ValidationError(validator_name="interface", ...)` on first
         mismatch.

    Why import the module: Python's reflection does not let us check method
    signatures from AST alone (`def forward(self, features, coords=None)` AST
    has no notion of "compatible with `def forward(self, features: Tensor,
    coords: Optional[Tensor] = None) -> AggregatorOutput`"). We import,
    inspect, and discard the class object.

    Why this isn't fully "static": pure-AST signature matching is a
    research-grade lift (covering type annotations, decorators, default
    values). Submitting code is operator-trusted; the validator's job is
    to catch HONEST mistakes (typo'd method name, wrong arg count) BEFORE
    queue, not to defend against malicious code (the consumer's git
    history is the trust root for that).
    """

    def __init__(self):
        from automil.registry.variants import LossVariant, ModelVariant, PolicyVariant
        self._abcs = {
            "model": ModelVariant,
            "loss": LossVariant,
            "policy": PolicyVariant,
        }

    def check(self, module_path: Path) -> None:
        """Validate a variant module. Raises ValidationError on failure."""
```

```python
class PurityValidator:
    """Static validator: no top-level I/O, network calls, or mutable globals.

    Pure AST walk — never imports the module. Inspects:
      1. Top-level `Call` nodes: rejects `open(...)`, `Path(...).write_*()`,
         `requests.*`, `urllib.*`, `socket.*`, `http.*`.
      2. Top-level `Assign` nodes: warns on mutable globals (lists, dicts,
         sets); constants (frozenset, tuple, str, int, float, bool, None,
         module-level functions, classes) are OK.
      3. Top-level `Expr` nodes: rejects function calls that aren't
         `register(...)` (the @register decorator is the only sanctioned
         module-level side effect — Plan 01-02's mechanism).
    """

    _BANNED_TOP_LEVEL_CALLS: frozenset[str] = frozenset({
        "open", "input", "exec", "eval", "compile",
    })
    _BANNED_TOP_LEVEL_MODULES: frozenset[str] = frozenset({
        "requests", "urllib", "socket", "http", "ftplib", "smtplib",
    })

    def check(self, module_path: Path) -> None:
        """Validate a variant module. Raises ValidationError on failure."""
```

    ```python
    """Static purity validator: rejects top-level I/O, network, mutable globals (REG-03 / D-30)."""
    from __future__ import annotations

    import ast
    import logging
    from pathlib import Path

    from automil.registry.errors import ValidationError

    logger = logging.getLogger(__name__)

    # Banned top-level call names (e.g., `open(...)`, `print(...)`, `exec(...)`)
    _BANNED_BUILTINS: frozenset[str] = frozenset({
        "open", "print", "input", "exec", "eval", "compile",
    })

    # Banned top-level module references — any `<mod>.foo(...)` or `from <mod> import ...`
    # followed by top-level use is rejected. Function-body usage is fine.
    _BANNED_MODULES: frozenset[str] = frozenset({
        "requests", "urllib", "socket", "http", "ftplib", "smtplib",
        "subprocess",
    })

    # Specific os.* attributes that imply side effects at top level.
    _BANNED_OS_ATTRS: frozenset[str] = frozenset({"system", "popen", "remove", "unlink", "mkdir", "rmdir"})


    def _is_immutable_literal(node: ast.AST) -> bool:
        """Return whether an AST expression is an immutable constant.

        Allowed: Constant, Name (refers to built-in like None, True, False; or
        a previously-imported name — we accept on faith), Tuple of immutables,
        BinOp on Constants (e.g., 3.14 * 2). Disallowed: List, Dict, Set,
        ListComp, DictComp, SetComp, GeneratorExp, Call (non-frozen).
        """
        if isinstance(node, ast.Constant):
            return True
        if isinstance(node, ast.UnaryOp):
            return _is_immutable_literal(node.operand)
        if isinstance(node, ast.Tuple):
            return all(_is_immutable_literal(elt) for elt in node.elts)
        if isinstance(node, ast.BinOp):
            return _is_immutable_literal(node.left) and _is_immutable_literal(node.right)
        if isinstance(node, ast.Name):
            # Only accept well-known immutable names.
            return node.id in {"None", "True", "False"}
        if isinstance(node, ast.Call):
            # Allow tuple(...), frozenset(...), Path(...) (Path is immutable-by-default).
            func = node.func
            if isinstance(func, ast.Name) and func.id in {"tuple", "frozenset"}:
                return True
            if isinstance(func, ast.Attribute) and isinstance(func.value, ast.Name):
                # e.g., os.path.join(...) — defer to "function call at module level"
                # which is rejected separately as a side-effect.
                return False
            return False
        return False


    class PurityValidator:
        """Static AST walk; never imports the module."""

        def check(self, module_path: Path) -> None:
            if not module_path.exists():
                raise ValidationError(
                    validator_name="purity",
                    path=module_path,
                    reason="module not found",
                    fix_suggestion="Verify the path exists and is readable.",
                )
            try:
                tree = ast.parse(module_path.read_text(), filename=str(module_path))
            except SyntaxError as e:
                raise ValidationError(
                    validator_name="purity",
                    path=module_path,
                    line=e.lineno,
                    column=e.offset,
                    reason=f"syntax error: {e.msg}",
                    fix_suggestion="Fix the Python syntax error in the module.",
                ) from e

            for node in tree.body:
                self._check_top_level_node(module_path, node)

        def _check_top_level_node(self, module_path: Path, node: ast.AST) -> None:
            # Allowed top-level constructs: imports, class / function defs,
            # immutable assignments, decorated class/function (incl. @register),
            # docstring (Expr with Constant str), `from __future__` imports,
            # ifs that are NOT __main__-blocks.
            if isinstance(node, (ast.Import, ast.ImportFrom)):
                return
            if isinstance(node, (ast.ClassDef, ast.FunctionDef, ast.AsyncFunctionDef)):
                return
            if isinstance(node, ast.Expr):
                # Module docstring is OK; bare top-level expressions are not.
                if isinstance(node.value, ast.Constant) and isinstance(node.value.value, str):
                    return
                # Bare top-level call — only `register(...)` is allowed (D-26 + Plan 01-02).
                # In practice the @register decorator lives on a ClassDef so we should
                # never see it as an Expr. Reject all top-level Expr-Call nodes.
                if isinstance(node.value, ast.Call):
                    self._reject_call(module_path, node.value, top_level=True)
                else:
                    raise ValidationError(
                        validator_name="purity",
                        path=module_path,
                        line=node.lineno,
                        column=node.col_offset,
                        reason=f"unexpected top-level expression: {ast.dump(node.value)[:60]}",
                        fix_suggestion="Move the expression into a function body or remove.",
                    )
                return
            if isinstance(node, (ast.Assign, ast.AnnAssign, ast.AugAssign)):
                value = getattr(node, "value", None)
                if value is None:
                    return  # bare AnnAssign (`x: int`) is metadata, OK
                if isinstance(value, ast.Call):
                    # mutable-via-Call: dict(), list(), set(), defaultdict(), etc.
                    self._reject_call(module_path, value, top_level=True)
                if isinstance(value, (ast.List, ast.Dict, ast.Set, ast.ListComp,
                                      ast.DictComp, ast.SetComp, ast.GeneratorExp)):
                    raise ValidationError(
                        validator_name="purity",
                        path=module_path,
                        line=node.lineno,
                        column=node.col_offset,
                        reason=(
                            f"mutable module-level global ({type(value).__name__}); "
                            "module-level state must be immutable (D-30)"
                        ),
                        fix_suggestion=(
                            "Use a tuple/frozenset for an immutable constant, or move "
                            "the mutable structure into a function/method body."
                        ),
                    )
                # Otherwise: literal value, OK.
                return
            if isinstance(node, ast.If):
                # Reject `if __name__ == "__main__":` blocks — variants are libraries.
                test = node.test
                is_main_block = (
                    isinstance(test, ast.Compare)
                    and isinstance(test.left, ast.Name)
                    and test.left.id == "__name__"
                    and len(test.ops) == 1
                    and isinstance(test.ops[0], ast.Eq)
                    and len(test.comparators) == 1
                    and isinstance(test.comparators[0], ast.Constant)
                    and test.comparators[0].value == "__main__"
                )
                if is_main_block:
                    raise ValidationError(
                        validator_name="purity",
                        path=module_path,
                        line=node.lineno,
                        column=node.col_offset,
                        reason="`if __name__ == \"__main__\":` block in a variant module (D-30)",
                        fix_suggestion=(
                            "Remove the `__main__` block — variants are libraries, "
                            "not scripts. If you need a runnable demo, put it in a "
                            "sibling tests/ file."
                        ),
                    )
                return  # other `if` blocks at module-level: pass for now (rare)
            # Default: any other top-level construct (Try, With, While, For, ...) is suspect.
            raise ValidationError(
                validator_name="purity",
                path=module_path,
                line=getattr(node, "lineno", None),
                column=getattr(node, "col_offset", None),
                reason=f"unsupported top-level construct: {type(node).__name__}",
                fix_suggestion="Remove or move into a function body.",
            )

        def _reject_call(self, module_path: Path, call: ast.Call, *, top_level: bool) -> None:
            func = call.func

            # Bare-name calls: open(), print(), exec(), eval(), compile(), input()
            if isinstance(func, ast.Name) and func.id in _BANNED_BUILTINS:
                raise ValidationError(
                    validator_name="purity",
                    path=module_path,
                    line=call.lineno,
                    column=call.col_offset,
                    reason=f"banned top-level builtin call: {func.id}()",
                    fix_suggestion=(
                        f"Move {func.id}() into a method body, or remove it. "
                        "Top-level I/O on import time leaks side effects to "
                        "every consumer that imports the module."
                    ),
                )

            # Attribute calls: requests.get(), urllib.request.urlopen(),
            # socket.socket(), os.system(), os.environ[...] = ..., subprocess.run().
            if isinstance(func, ast.Attribute):
                # Walk to the leftmost Name to find the module.
                attr_chain: list[str] = []
                cur: ast.AST = func
                while isinstance(cur, ast.Attribute):
                    attr_chain.append(cur.attr)
                    cur = cur.value
                if isinstance(cur, ast.Name):
                    root = cur.id
                    full = ".".join(reversed([root] + attr_chain))

                    if root in _BANNED_MODULES:
                        raise ValidationError(
                            validator_name="purity",
                            path=module_path,
                            line=call.lineno,
                            column=call.col_offset,
                            reason=f"banned top-level network/process call: {full}(...)",
                            fix_suggestion=(
                                f"Move {full}(...) into a method body. "
                                "Top-level network or process side effects on import "
                                "violate D-30 module-level purity."
                            ),
                        )
                    if root == "os" and attr_chain and attr_chain[-1] in _BANNED_OS_ATTRS:
                        raise ValidationError(
                            validator_name="purity",
                            path=module_path,
                            line=call.lineno,
                            column=call.col_offset,
                            reason=f"banned top-level os.* call: {full}(...)",
                            fix_suggestion=(
                                f"Move {full}(...) into a method body."
                            ),
                        )

            # Path("/x").write_text("y") — the call is a method on a Path instance;
            # the call.func is Attribute(.attr="write_text", value=Call(Path, ...)).
            if isinstance(func, ast.Attribute) and func.attr.startswith(("write", "read", "append")):
                # Conservative: any top-level write_*/read_*/append_* call is a
                # filesystem/I/O smell.
                raise ValidationError(
                    validator_name="purity",
                    path=module_path,
                    line=call.lineno,
                    column=call.col_offset,
                    reason=f"banned top-level filesystem/I/O call: .{func.attr}(...)",
                    fix_suggestion="Move the I/O into a method body.",
                )

            # os.environ["X"] = "y" — this is a Subscript Assign, NOT a Call. Caught
            # in _check_top_level_node's Assign handler (we call _reject_call on

            # default reject branch.). For now, we explicitly catch
            # `os.environ.<setter>` if it ever shows up as a Call.
    ```

    ```python
    def _abcs() -> dict[str, type]:
        """Lazy import to avoid circular at module import time."""
        from automil.registry.variants import LossVariant, ModelVariant, PolicyVariant
        return {"model": ModelVariant, "loss": LossVariant, "policy": PolicyVariant}


    def _required_methods(kind: str) -> list[str]:
        """The methods the ABC marks @abstractmethod for each kind."""
        return {
            "model": ["forward"],
            "loss": ["__call__"],
            "policy": ["wrap_optimizer"],
        }[kind]


    def _find_register_calls(tree: ast.Module) -> list[tuple[ast.ClassDef, ast.Call]]:
        """Return [(class_def_ast, register_call_ast), ...] for every @register-decorated class.

        D-26: each variant module should contain exactly one. The validator's job
        is to enforce that.
        """
        out: list[tuple[ast.ClassDef, ast.Call]] = []
        for node in tree.body:
            if not isinstance(node, ast.ClassDef):
                continue
            for dec in node.decorator_list:
                if isinstance(dec, ast.Call):
                    func = dec.func
                    if isinstance(func, ast.Name) and func.id == "register":
                        out.append((node, dec))
                    elif isinstance(func, ast.Attribute) and func.attr == "register":
                        out.append((node, dec))
        return out


    def _extract_kind_from_register_call(call: ast.Call) -> Optional[str]:
        """The @register call has VariantSpec(...) as its first arg; find kind=...".

        Returns None if we can't statically determine the kind (e.g., kind is a
        variable instead of a literal). In that case we defer to runtime
        introspection after import.
        """
        if not call.args:
            return None
        spec_call = call.args[0]
        if not isinstance(spec_call, ast.Call):
            return None
        for kw in spec_call.keywords:
            if kw.arg == "kind" and isinstance(kw.value, ast.Constant):
                v = kw.value.value
                if isinstance(v, str):
                    return v
        return None


    def _import_module_from_path(module_path: Path) -> Any:
        """Import a Python file as a module without polluting sys.modules permanently."""
        spec = importlib.util.spec_from_file_location(
            f"_automil_validator_{module_path.stem}", module_path
        )
        if spec is None or spec.loader is None:
            raise ValidationError(
                validator_name="interface",
                path=module_path,
                reason="cannot create module spec",
                fix_suggestion="Verify the file is a valid Python module.",
            )
        module = importlib.util.module_from_spec(spec)
        spec.loader.exec_module(module)  # type: ignore[union-attr]
        return module


    def _signature_compatible(abc_method: Any, variant_method: Any) -> tuple[bool, str]:
        """Check that variant_method's signature is compatible with abc_method's.

        Compatible = same parameter names (excluding self) for required params,
        and variant may add MORE optional params. Return (ok, reason_if_not_ok).
        """
        try:
            abc_sig = inspect.signature(abc_method)
            var_sig = inspect.signature(variant_method)
        except (ValueError, TypeError) as e:
            return False, f"could not inspect signature: {e}"

        abc_params = list(abc_sig.parameters.values())
        var_params = list(var_sig.parameters.values())

        # Skip `self` for both.
        if abc_params and abc_params[0].name == "self":
            abc_params = abc_params[1:]
        if var_params and var_params[0].name == "self":
            var_params = var_params[1:]

        # Check: every required (no default) param in abc must exist in var.
        for i, p in enumerate(abc_params):
            if p.default is inspect.Parameter.empty and i >= len(var_params):
                return False, f"missing parameter '{p.name}'"
            if i < len(var_params) and var_params[i].name != p.name:
                # We accept positional renaming only if name kw-compatibility
                # holds — but in practice, parameter renaming is a smell.
                # Be permissive: accept if positional count matches.
                pass
        return True, ""


    class InterfaceValidator:
        """ABC subclass + required-method signature check.

        Run order: AST scan first (cheap fail on no-@register / multiple @register
        / syntax error), then dynamic import + reflection.
        """

        def check(self, module_path: Path) -> None:
            if not module_path.exists():
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    reason="module not found",
                    fix_suggestion="Verify the path exists and is readable.",
                )

            try:
                source = module_path.read_text()
                tree = ast.parse(source, filename=str(module_path))
            except SyntaxError as e:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=e.lineno,
                    column=e.offset,
                    reason=f"syntax error: {e.msg}",
                    fix_suggestion="Fix the Python syntax error.",
                ) from e

            register_classes = _find_register_calls(tree)
            if not register_classes:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    reason="no @register decorator found; not a variant module",
                    fix_suggestion=(
                        "Decorate the variant class with @register(VariantSpec(...)). "
                        "See src/automil/registry/registrar.py for the API."
                    ),
                )
            if len(register_classes) > 1:
                first_class, _ = register_classes[0]
                second_class, _ = register_classes[1]
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=second_class.lineno,
                    column=second_class.col_offset,
                    reason=(
                        f"multiple @register-decorated classes: {first_class.name!r} "
                        f"AND {second_class.name!r}. D-26 requires a single .py file "
                        "per variant; if too big to fit one file, it's two variants."
                    ),
                    fix_suggestion=(
                        f"Split into two files: one for {first_class.name} and one "
                        f"for {second_class.name}."
                    ),
                )

            class_def, register_call = register_classes[0]
            kind = _extract_kind_from_register_call(register_call)

            # Import the module to inspect the actual class object.
            try:
                module = _import_module_from_path(module_path)
            except SyntaxError as e:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=e.lineno,
                    reason=f"syntax error during import: {e.msg}",
                    fix_suggestion="Fix the Python syntax error.",
                ) from e
            except ValidationError:
                raise
            except Exception as e:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    reason=f"import failed: {type(e).__name__}: {e}",
                    fix_suggestion="Inspect the import-time error in the module.",
                ) from e

            cls = getattr(module, class_def.name, None)
            if cls is None:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=class_def.lineno,
                    reason=f"class {class_def.name!r} not found after import",
                    fix_suggestion="Confirm the class is defined at module top level.",
                )

            abcs = _abcs()
            if kind is None:
                # Fall back: introspect bases.
                for k, abc in abcs.items():
                    if isinstance(cls, type) and issubclass(cls, abc):
                        kind = k
                        break

            if kind not in abcs:
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=class_def.lineno,
                    reason=f"unknown kind {kind!r}",
                    fix_suggestion=(
                        "Set kind='model' | 'loss' | 'policy' in the VariantSpec."
                    ),
                )

            abc_class = abcs[kind]
            if not (isinstance(cls, type) and issubclass(cls, abc_class)):
                raise ValidationError(
                    validator_name="interface",
                    path=module_path,
                    line=class_def.lineno,
                    reason=(
                        f"class {class_def.name!r} declared kind={kind!r} but is not "
                        f"a subclass of {abc_class.__name__}"
                    ),
                    fix_suggestion=(
                        f"Change `class {class_def.name}({abc_class.__name__}):` "
                        f"or update kind in VariantSpec."
                    ),
                )

            # Required-method existence + signature compatibility.
            required = _required_methods(kind)
            for method_name in required:
                variant_method = getattr(cls, method_name, None)
                # If the class defines the method but it's still abstract,
                # `cls()` would raise TypeError. We check directly.
                if (
                    variant_method is None
                    or getattr(variant_method, "__isabstractmethod__", False)
                ):
                    raise ValidationError(
                        validator_name="interface",
                        path=module_path,
                        line=class_def.lineno,
                        reason=(
                            f"missing required method {method_name!r} on {class_def.name!r} "
                            f"(kind={kind} requires {required})"
                        ),
                        fix_suggestion=(
                            f"Add `def {method_name}(self, ...)` to the class. "
                            f"See {abc_class.__name__}.{method_name} for the expected signature."
                        ),
                    )
                abc_method = getattr(abc_class, method_name)
                ok, reason = _signature_compatible(abc_method, variant_method)
                if not ok:
                    raise ValidationError(
                        validator_name="interface",
                        path=module_path,
                        line=class_def.lineno,
                        reason=(
                            f"method {method_name!r} signature incompatible with "
                            f"{abc_class.__name__}.{method_name}: {reason}"
                        ),
                        fix_suggestion=(
                            f"Match the ABC's signature: see {abc_class.__name__}.{method_name}."
                        ),
                    )
    ```
