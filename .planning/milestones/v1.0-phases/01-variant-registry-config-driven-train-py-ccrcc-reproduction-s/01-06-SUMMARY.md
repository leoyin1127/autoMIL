## Idempotence Proof

`regenerate_init_py` separates the timestamp onto its own comment line:

```
# AUTO-GENERATED by `automil refresh-registry`. DO NOT EDIT.
# Re-run the command after adding or renaming a variant module.
# generated-at: 2026-05-02T10:00:00.000000+00:00    <-- stripped during compare
from . import clam_mb_v0176  # noqa: F401
from . import clam_mb_v0188  # noqa: F401
```

Stripping lines starting with `# generated-at:` produces byte-identical output on consecutive runs when directory contents have not changed. Covered by `test_regenerate_init_py_idempotent_body`.
